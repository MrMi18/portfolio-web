ByteCrew - NammaSuraksha (AI Scam &Phishing Detection System)

April 1, 2025 - Now

NammaSuraksha is a cross-platform AI-powered scam and phishing detection system developed during the NammaSuraksha Hackathon 2025 by Team ByteCrew. It intelligently detects malicious links and scam messages across web pages, emails, instant messaging platforms, social media, and SMS using OpenAI 's GPT-4. The project features a centralized backend with Express.js and PostgreSQL, a Next.js-based web dashboard, a React Native mobile app for real-time SMS scanning, and a Chrome extension for live phishing alerts. The system delivers immediate alerts and offers educational resources to raise awareness, helping users stay safe online.
